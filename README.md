# Pink as Fox

A theme for Visual Studio Code based on the Dark+ builtin theme.

![screenshot](res/screenshot0.png)

## Font Recommendation

[Fira Code](https://github.com/tonsky/FiraCode)

## Settings Recommendation

```jsonc
{
    "workbench.tree.renderIndentGuides": "none",
}
```
